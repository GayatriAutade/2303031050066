## Notification App Backend
This project is a simple frontend application built using **React + Vite** that fetches and displays notifications from a backend API.

The goal of this project is to demonstrate:
- API integration
- Component-based architecture
- State management using React Hooks
- Clean UI rendering of dynamic data

##  Features

- Fetch notifications from external API
- Display notifications in a structured card layout
- Loading state while data is being fetched
- Empty state handling when no data is available
- Modular and reusable component structure
- Basic filtering-ready architecture (extensible)
